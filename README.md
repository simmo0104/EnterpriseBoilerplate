# Enterprise Boilerplate

A production-inspired, assessment-ready enterprise boilerplate built with ASP.NET Core (.NET 8), Angular 20, and SQL Server.

## Purpose

This repository exists to eliminate repetitive setup time during technical assessments and personal projects, while demonstrating production-quality engineering practices.

## Tech Stack

- **Backend:** ASP.NET Core (.NET 8), ADO.NET, SQL Server, Stored Procedures
- **Frontend:** Angular 20, TypeScript, Standalone Components
- **Testing:** xUnit, FluentAssertions, Moq
- **Database:** SQL Server, TVPs, Stored Procedures

## Structure

| Folder | Purpose |
|--------|---------|
| `backend/` | ASP.NET Core Web API solution |
| `frontend/` | Angular 20 application |
| `database/` | SQL scripts, stored procedures, TVPs |
| `docs/` | Architecture notes, PR templates, conventions |
